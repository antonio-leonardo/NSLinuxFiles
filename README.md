# 🎮 NSLinuxFiles: Otimização Extrema para Nintendo Switch

Este repositório automatiza a configuração do L4T Linux para emulação de alta performance.

## 🚀 Instalação Rápida (One-Liner)
Abra o terminal no seu Switch e cole o comando abaixo:
```bash
curl -sSL [https://raw.githubusercontent.com/antonio-leonardo/NSLinuxFiles/main/setup.sh](https://raw.githubusercontent.com/antonio-leonardo/NSLinuxFiles/main/setup.sh) | bash
